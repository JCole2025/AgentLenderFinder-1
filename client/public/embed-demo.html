Agent Finder Widget - Embedding Instructions

This demo shows how to embed the BiggerPockets Agent Finder widget on your website. There are two embedding options: direct embed and modal popup.

Direct Embed
Modal Popup
Configuration Options

Direct Embed

Embed the Agent Finder widget directly on your page:

<!-- 1. Add a container for the widget --> <div id="bp-agent-finder-widget"></div> <!-- 2. Include the embed script --> <script src="https://your-domain.com/embed.js"></script>

Live Example:

Modal Popup

Use a button to trigger the Agent Finder widget in a modal:

<!-- 1. Add a container for the button --> <div id="bp-agent-finder-widget"></div> <!-- 2. Include the embed script with modal mode enabled --> <script src="https://your-domain.com/embed.js" data-modalmode="true" data-buttontext="Find a Real Estate Agent"></script>

Live Example:

Configuration Options

The widget can be customized using data attributes on the script tag:

Option Description Default Example
data-containerid ID of the container element bp-agent-finder-widget data-containerid="my-custom-container"
data-width Width of the widget 100% data-width="800px"
data-height Height of the widget (direct embed only) 650px data-height="500px"
data-primarycolor Primary color (hex code) #3b82f6 data-primarycolor="#ff6600"
data-buttontext Text for the trigger button (modal mode only) Find an Agent data-buttontext="Connect with an Agent"
data-modalmode Whether to show as modal popup false data-modalmode="true"

Example with all options:

<script src="https://your-domain.com/embed.js" data-containerid="custom-finder" data-width="100%" data-height="700px" data-primarycolor="#ff6600" data-buttontext="Find Real Estate Agents" data-modalmode="true"></script>
