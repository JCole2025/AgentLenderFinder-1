BiggerPockets Agent Finder

Embedding API Documentation

Overview

The BiggerPockets Agent Finder can be embedded on any website using a simple script tag. This document provides details on all available endpoints for embedding and interacting with the widget.

API Endpoints

Endpoint Description Usage
/embed.js The main embedding script that creates and initializes the widget iframe Add to any HTML page via a script tag
/embed The embedded version of the agent finder form Automatically loaded by embed.js (do not use directly)
/example.html Example page showing embedding options and documentation Reference for developers
/test-embed.html Test page for the embedded widget with debugging controls Testing and development

Important Note

When embedding the widget, always use the absolute URL to the embed.js script:

https://your-replit-domain.com/embed.js

This ensures the widget loads correctly across different domains.

Basic Implementation

To embed the Agent Finder on your website, add the following code:

<script 
  src="https://your-replit-domain.com/embed.js" 
  data-containerid="bp-agent-finder"
  data-width="100%"
  data-height="700px"
  data-borderradius="8px"
  data-boxshadow="0 4px 12px rgba(0, 0, 0, 0.1)">
</script>

Replace your-replit-domain.com with your actual Replit domain.

Configuration Options

Attribute Description Default
data-containerid ID for the container element "bp-agent-finder"
data-width Width of the widget "100%"
data-height Height of the widget "650px"
data-border Border style "none"
data-borderradius Border radius "8px"
data-boxshadow Box shadow "0 4px 12px rgba(0, 0, 0, 0.1)"
data-scrolling Enable/disable iframe scrolling "no"
data-allowfullscreen Allow fullscreen mode "false"

JavaScript API

The embed script exposes a JavaScript API for programmatically interacting with the widget. This API is accessible through the window.BiggerPocketsAgentFinder object. 

// Get reference to the widget instance
const widget = window.BiggerPocketsAgentFinder.instances[Object.keys(window.BiggerPocketsAgentFinder.instances)[0]];

// Reset the form
widget.reset();

// Get the current form state
widget.getState();

Event Handling

You can listen for events from the widget using the window's message event. This allows you to respond to form state changes, submissions, etc. 

window.addEventListener('message', function(event) {
  // Verify the origin of the message
  if (event.origin !== 'https://your-replit-domain.com') return;
  
  const data = event.data;
  
  // Handle different message types
  switch (data.type) {
    case 'resize':
      console.log('Widget resized to height:', data.height);
      break;
      
    case 'stepChange':
      console.log('Form step changed to:', data.step);
      break;
      
    case 'formSubmitted':
      console.log('Form successfully submitted!');
      // You can trigger analytics events or redirects here
      break;
  }
});
