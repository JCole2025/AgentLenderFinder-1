BiggerPockets Agent Finder Widget

How to Embed the Widget

To embed the BiggerPockets Agent Finder Widget on your website, simply add the following code:

<script 
  src="https:///embed.js" 
  data-containerid="bp-agent-finder"
  data-width="100%"
  data-height="650px"
  data-borderradius="8px"
  data-boxshadow="0 4px 12px rgba(0, 0, 0, 0.1)">
</script>

Customization Options

You can customize the widget by adding the following data attributes to the script tag:

  • data-containerid: ID for the container element (default: "bp-agent-finder")
  • data-width: Width of the widget (default: "100%")
  • data-height: Height of the widget (default: "650px")
  • data-border: Border style (default: "none")
  • data-borderradius: Border radius (default: "8px")
  • data-boxshadow: Box shadow (default: "0 4px 12px rgba(0, 0, 0, 0.1)")
